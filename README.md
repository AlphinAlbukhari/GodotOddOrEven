# GodotOddOrEven
Godot script to check if a number is odd or even

# Usage
Just put the script into your project folder and there will be OddEven class.
|   Method   |        Description        |
| ---------- | ------------------------- |
| is_odd(x)  | Returns true if x is odd  |
| is_even(x) | Returns true if x is even |
